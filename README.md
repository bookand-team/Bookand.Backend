### 독립서점 서비스 
BOOK&는 수도권에 있는 독립서점의 정보를 제공하며, 맞춤형 큐레이션 피드 제공을 통해, 독립서점을 소개합니다.

## 💡 Features

- 소셜 로그인 (구글, 애플)
- 관리자 로그인
- 위치 기반 독립 서점에 대한 정보 조회
- 새로운 서점에 대한 제보, 추가
- 서점, 책, 인터뷰 내용을 담은 아티클
- 권한에 따른 아티클 작성, 조회

## ⚒️ Tech Stack

- Spring Boot (Java 17)
- Spring Data JPA
- Spring Security
- PostgreSQL
- QueryDSL
- AWS EC2, RDS, S3, CloudFront
- Github Actions
